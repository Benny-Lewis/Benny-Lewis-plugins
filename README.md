# Benny-Lewis-plugins

Claude Code plugin marketplace.

## Install

```
/plugin marketplace add Benny-Lewis/Benny-Lewis-plugins
```

Then browse and install plugins:

```
/plugin
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [home-assistant-assistant](https://github.com/Benny-Lewis/home-assistant-assistant) | Manage Home Assistant through natural language — automations, scripts, scenes, dashboards, naming, deployment |
| [garden-bot](https://github.com/Benny-Lewis/garden-bot) | Gardening and landscaping assistant — persistent profiles, expert plant advice, visual landscape design |
